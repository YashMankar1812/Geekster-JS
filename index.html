About FunVilla

FunVilla is a creative playground built by Yash Mankar, where learning meets fun. This platform showcases interactive JavaScript projects, mini-games, UI experiments, and DOM-based applications designed to strengthen frontend skills while keeping things enjoyable.

Every project inside FunVilla focuses on clean UI, smooth interaction, and real-world JavaScript logic — making it perfect for learners, recruiters, and curious minds.
